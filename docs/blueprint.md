# **App Name**: Java Student Manager

## Core Features:

- Student Registration: Form for registering new students with fields for name, address, contact information, and any other relevant details.
- Student Database: Store student data in a MySQL database using a Java backend.
- Student Search: Implement a search feature to quickly find students based on various criteria (name, ID, etc.).
- Student Profile: Dedicated page for each student with all available information and options to edit/update.
- Student Roster Display: View the entire roster of students using pages. Show a maximum of 20 students per page. Display limited student information such as name, student id, and email.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and stability.
- Background color: Light blue (#E8EAF6) for a calm and clean interface.
- Accent color: Amber (#FFC107) for highlights and important actions.
- Body and headline font: 'PT Sans' for clear and readable text.
- Use simple, clear icons from a library like Material Design Icons to represent actions and categories.
- Use a clean, organized layout with consistent spacing and alignment.
- Subtle transitions and animations to improve user experience.
